Отчет "Отчет по банковским операциям"


Общая информация


В отчете Отчет по банковским операциям отображаются поступления и расходы денежных средств по расчетным счетам. Движения показываются в разрезе вида движений (банковских категорий), банков, расчетных счетов, статей движения денежных средств, счетов и корреспондирующих счетов бухгалтерского учета, получателей и плательщиков, документов оплаты.

Значение среднего остатка денежных средств рассчитывается как сумма остатков на начало каждого дня, деленное на количество дней.

Информация для бухгалтера


Поступление денежных средств определяется как оборот по дебету счетов 51, 52, 55 за период. Вид движения денежных средств определяется по счету, который корреспондирует со счетом денежных средств. Например, по виду движения Поступления от контрагентов отражаются проводки по дебету счетов денежных средств и по кредиту счетов с субконто Контрагенты, исключая (57.01, 57.21, 57.02, 57.22, 57.03, 68.22, 68.32, 68.42, 76.АВ, 76.ВА, 76.К, 76.Н, 76.ОТ). 

Расход денежных средств определяется как оборот по кредиту счетов 51, 52, 55 за период. Вид движения денежных средств определяется по счету, который корреспондирует со счетом денежных средств. Например, по виду движения Выдача наличных отражаются проводки по кредиту счетов денежных средств и по дебету кассовых счетов (субсчета 50).
