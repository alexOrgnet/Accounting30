Документ "Прекращение аренды (лизинга)"

Документ предназначен для отражения в учете прекращения договора аренды (лизинга).

Документ может использоваться для следующих хозяйственных операций:

  • Досрочное прекращение договора аренды
  • Прекращение договора аренды по окончании срока
  • Досрочное прекращение договора лизинга с возвратом лизингодателю предмета лизинга.

Печатные формы

Для документа предусмотрены следующие печатные формы:

  • Акт о списании основных средств (произвольная форма).
  • Акт о списании объекта основных средств (кроме автотранспортных средств) ОС-4.
  • Акт о списании автотранспортных средств ОС-4а.
  • Акт о списании групп объектов основных средств ОС-4б. Формируется при списании более чем одного основного средства.
