Документ "Изменение элементов амортизации ОС"

Документ предназначен для изменения элементов амортизации основных средств.

При вводе документа в шапке нужно указать следующие реквизиты:

  • Событие - наименование события в жизни основного средства, которое отражается документом.
  • Отразить в - устанавливается в зависимости от того, в каком из учетов необходимо изменить элементы  амортизации.

В табличном поле Основные средства указывается список основных средств, у которых изменяются элементы начисления амортизации.

  • Основное средство - основные средства, у которых изменяются параметры начисления амортизации. 
  • Остальные колонки заполняются текущими элементами амортизации на основании данных информационной базы.
  • Если значение элемента амортизации не изменяется, то "До изменения" и "После изменения" будут равны.
  • Состав колонок может отличаться в зависимости от способа начисления амортизации.
  • Способ начисления амортизации - Способ начисления амортизации в учете, например, линейный.
  • Срок полезного использования - срок полезного использования основного средства.
  • Остаток срока полезного использования для начисления амортизации. Это вычисляемое поле на основании общего срока полезного использования и фактически использованного.
  • Коэффициент ускорения - если амортизация по основному средству начисляется способом уменьшаемого остатка, указывается коэффициент ускорения.
  • Объем работ - если амортизация по основному средству начисляется пропорционально объему продукции (работ), указывается предполагаемый объем продукции (работ), на выработку которого рассчитано основное средство.
  • Остаток работ - если амортизация по основному средству начисляется пропорционально объему продукции (работ), указывается остаток предполагаемого объема продукции (работ) для начисления амортизации.
  • Ликвидационная стоимость - расчетная сумма, которую организация получила бы на текущий момент от выбытия актива после вычета расчетных затрат на выбытие.
  • Компоненты - если ведется обособленный учет существенных по величине затрат на проведение ремонта, технического осмотра, технического обслуживания основных средств, указывается состав компонентов амортизации и параметры начисления амортизации по ним. Возможность настройки компонентов амортизации предусмотрена в 1С:Бухгалтерии КОРП.

    • Примечание
    Для организаций, уплачивающих налог на прибыль, в колонке Срок полезного использования (НУ) указывается срок полезного использования для начисления амортизации. Если срок полезного использования не изменяется, в этой колонке указывается прежний срок полезного использования.
